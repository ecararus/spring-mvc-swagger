spring-mvc-swagger

Intention: Write a simple Spring MVC application and supply documentation for REST API

Prove that Swagger can be integrated with SpringMVC when configuration is done via XML.

System requirements:

  • Java 8
  • Free port 8080

How to start and use

    mvn clean install jetty:run
  • Open browser visit localhost:8080/
